# Notes

Personal notes maintained collaboratively with Claude. Rendered in Obsidian. See `CLAUDE.md` for structure and navigation, `CONVENTIONS.md` for authoring conventions.

## Structure

- **`wiki/`** — All entries, organized by category (`books/`, `movies/`, `products/`, `trips/`, etc.) and projects (`projects/`).
- **`lists/`** — Browsing views (one `.md` per category with embedded Bases).
- **`templates/`** — One template per entry category.
- **`sources/`** — Raw source material archive (immutable after capture).
- **`log/`** — Session recaps with learnings and decisions.
- **`wiki-index.md`** — Directory of all wiki entries by category.
- **`sessions-index.md`** — Session directory with dates and summaries (newest-first).
- **`wiki-changelog.md`** — Append-only record of wiki changes (newest-first).
