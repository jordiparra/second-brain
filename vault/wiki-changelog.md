# Wiki changelog

Append-only record of wiki changes. Newest-first — new entries are prepended to the top. Existing entries are never modified.

Format: one paragraph per **cohesive change thread**, blank line between entries. Lead with the date in bold, an em dash, then a capitalized verb (Ingested, Updated, Added, Fixed, Renamed, Created) describing what changed and why. A single thread of work that touches many files across several tool calls is one entry — enumerate affected files inside the paragraph. Unrelated changes in the same session get separate entries.

**YYYY-MM-DD** — Capitalized verb describing what changed and why.
